## 详细登录流程

### 1. 账号密码登录 (支持手机号/邮箱/账号)

```
┌─────────────────────────────────────────────────────────────────┐
│                    流程图                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  用户输入账号 (手机号/邮箱/用户名) + 密码                       │
│           ↓                                                     │
│  自动识别账号类型：                                             │
│    • 手机号正则匹配 → 按手机号查询                              │
│    • 邮箱正则匹配 → 按邮箱查询                                  │
│    • 其他 → 按用户名查询                                        │
│           ↓                                                     │
│  验证密码                                                       │
│           ↓                                                     │
│  登录成功？                                                     │
│    ├─ ✅ 是 → 颁发 Token，跳转首页                              │
│    └─ ❌ 否 → 记录失败次数                                      │
│              ↓                                                  │
│         失败次数 ≥ 1？                                          │
│           ├─ ✅ 是 → 显示图形验证码                             │
│           │         ↓                                           │
│           │       输入验证码                                    │
│           │         ↓                                           │
│           │       验证码正确？                                  │
│           │         ├─ ✅ 是 → 重新验证密码                     │
│           │         └─ ❌ 否 → 提示错误，刷新验证码             │
│           └─ ❌ 否 → 提示"账号或密码错误"                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**业务规则**:
- 账号类型识别：
  - 手机号：`/^1[3-9]\d{9}$/`
  - 邮箱：`/^[^\s@]+@[^\s@]+\.[^\s@]+$/`
  - 其他：用户名
- 密码错误 1 次后显示图形验证码
- 连续失败 5 次，锁定账号 15 分钟
- 账号 (手机号/邮箱/用户名) 必须已注册
- 用户 ID 使用 UUID

### 2. 手机号 + 短信验证码登录

```
┌─────────────────────────────────────────────────────────────────┐
│                    流程图                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  用户输入手机号                                                 │
│           ↓                                                     │
│  验证格式 (手机号正则)                                          │
│           ↓                                                     │
│  显示图形拖拽滑块验证码                                         │
│           ↓                                                     │
│  滑块验证成功？                                                 │
│    ├─ ❌ 否 → 提示"验证失败"，刷新滑块                          │
│    └─ ✅ 是 → 发送短信验证码                                    │
│              ↓                                                  │
│         输入短信验证码                                          │
│           ↓                                                     │
│         验证码正确？                                            │
│           ├─ ✅ 是 → 检查手机号是否注册                         │
│           │         ├─ 已注册 → 登录成功，颁发 Token            │
│           │         └─ 未注册 → 自动注册 → 登录成功             │
│           └─ ❌ 否 → 提示"验证码错误"                           │
│                                                                 │
│  短信验证码规则：                                               │
│  • 有效期：5 分钟                                                │
│  • 重发间隔：60 秒                                               │
│  • 每日上限：10 条/手机号                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**业务规则**:
- 滑块验证：极验/易点云等第三方服务
- 短信服务商：阿里云短信/腾讯云短信
- 自动注册：生成随机密码，用户后续可设置
- 手机号必须唯一
- 用户 ID 使用 UUID

### 4. 微信扫码登录

```
┌─────────────────────────────────────────────────────────────────┐
│                    流程图                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  用户点击"微信扫码登录"                                         │
│           ↓                                                     │
│  生成微信二维码 (带临时 ticket)                                 │
│           ↓                                                     │
│  用户微信扫码并确认                                              │
│           ↓                                                     │
│  微信回调 (携带 code)                                           │
│           ↓                                                     │
│  使用 code 换取微信 OpenID + UserInfo                           │
│           ↓                                                     │
│  检查 OpenID 是否绑定账号                                        │
│    ├─ 已绑定 → 登录成功，颁发 Token                             │
│    └─ 未绑定 → 自动注册新账号 → 绑定 OpenID → 登录成功          │
│                                                                 │
│  微信配置：                                                     │
│  • 微信开放平台账号                                              │
│  • AppID + AppSecret                                            │
│  • 回调域名配置                                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**业务规则**:
- 微信开放平台认证
- OpenID 与账号绑定关系存储
- 自动注册：使用微信昵称 + 随机字符生成用户名
- 用户 ID 使用 UUID

## 功能需求

### 输入验证

| 字段 | 类型 | 验证规则 | 错误提示 |
|------|------|----------|----------|
| 手机号 | string | `/^1[3-9]\d{9}$/` | "请输入正确的手机号" |
| 邮箱 | string | 标准邮箱正则 | "请输入正确的邮箱" |
| 密码 | string | 6-20 位，至少包含字母和数字 | "密码格式不正确" |
| 验证码 | string | 4 位数字/字母 | "验证码不正确" |
| 短信验证码 | string | 6 位数字 | "验证码不正确" |
| 邮件验证码 | string | 6 位数字 | "验证码不正确" |
| 滑块验证 | token | 第三方验证通过 | "滑块验证失败" |

---

### 业务规则

```
1. 唯一性检查
   • 注册时检查手机号/邮箱/用户名是否已存在
   • 登录时检查账号是否存在

2. 安全策略
   • 密码错误 1 次 → 显示图形验证码
   • 密码错误 5 次 → 锁定账号 15 分钟
   • 短信验证码 → 滑块验证后才能发送
   • 邮件验证码 → 滑块验证后才能发送

3. 自动注册
   • 手机号未注册 → 自动生成账号 + 随机密码
   • 邮箱未注册 → 自动生成账号 + 随机密码
   • 微信未绑定 → 自动生成账号 (微信昵称 + 随机字符)

4. Token 管理
   • Access Token: 2 小时
   • Refresh Token: 7 天
   • 支持多设备登录

5. 会话管理
   • 支持单点登录 (SSO)
   • 支持多设备同时在线
   • 支持设备管理 (查看/踢出设备)
```
