# SaaS 平台项目

> 模块化 SaaS 平台，多前端 + 统一后端架构

**版本**: 1.0.0  
**创建时间**: 2026-03-28  
**仓库**: https://github.com/qmuyangren/saas-platform

---

## 项目简介

基于 Node.js + React 的模块化 SaaS 平台，采用 **多前端 + 统一后端** 架构：
- 统一认证中心 (SSO)
- 多租户架构
- 多业务前端 (电商/OA/CMS/党建/校园等)
- 统一后端服务
- AI 能力集成
- MCP 工具提供

## 快速开始

### admin-frontend (后台管理前端)

```bash
# 进入项目目录
cd admin-frontend

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 访问 http://localhost:5173
```

### server (统一后端服务)

```bash
# 进入项目目录
cd server

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 访问 http://localhost:3000
```
