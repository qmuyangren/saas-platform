## 详细登录流程

### 1. 账号密码登录 (支持手机号/邮箱/账号)

```
┌─────────────────────────────────────────────────────────────────┐
│                    流程图                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  用户输入账号 (手机号/邮箱/用户名) + 密码                       │
│           ↓                                                     │
│  自动识别账号类型：                                             │
│    • 手机号正则匹配 → 按手机号查询                              │
│    • 邮箱正则匹配 → 按邮箱查询                                  │
│    • 其他 → 按用户名查询                                        │
│           ↓                                                     │
│  验证密码                                                       │
│           ↓                                                     │
│  登录成功？                                                     │
│    ├─ ✅ 是 → 颁发 Token，跳转首页                              │
│    └─ ❌ 否 → 记录失败次数                                      │
│              ↓                                                  │
│         失败次数 ≥ 1？                                          │
│           ├─ ✅ 是 → 显示图形验证码                             │
│           │         ↓                                           │
│           │       输入验证码                                    │
│           │         ↓                                           │
│           │       验证码正确？                                  │
│           │         ├─ ✅ 是 → 重新验证密码                     │
│           │         └─ ❌ 否 → 提示错误，刷新验证码             │
│           └─ ❌ 否 → 提示"账号或密码错误"                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**业务规则**:
- 账号类型识别：
  - 手机号：`/^1[3-9]\d{9}$/`
  - 邮箱：`/^[^\s@]+@[^\s@]+\.[^\s@]+$/`
  - 其他：用户名
- 密码错误 1 次后显示图形验证码
- 连续失败 5 次，锁定账号 15 分钟
- 账号 (手机号/邮箱/用户名) 必须已注册
- 用户 ID 使用 UUID

---

### 2. 手机号 + 短信验证码登录

**业务规则**:
- 滑块验证：极验/易点云等第三方服务
- 短信服务商：阿里云短信/腾讯云短信
- 自动注册：生成随机密码，用户后续可设置
- 手机号必须唯一
- 用户 ID 使用 UUID

---

### 3. 邮箱 + 邮箱验证码登录

**业务规则**:
- 邮箱格式验证：标准邮箱正则
- 邮件服务商：SendGrid/阿里云邮件推送
- 自动注册：生成随机密码，用户后续可设置
- 邮箱必须唯一
- 用户 ID 使用 UUID

---

### 4. 微信扫码登录

**业务规则**:
- 微信开放平台认证
- OpenID 与账号绑定关系存储
- 自动注册：使用微信昵称 + 随机字符生成用户名
- 用户 ID 使用 UUID

---

### 5. 后台管理登录 ⭐

```
┌─────────────────────────────────────────────────────────────────┐
│                    后台管理登录流程                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ⚠️ 重要：后台管理登录无自动注册！                               │
│                                                                 │
│  登录方式：                                                     │
│  • 用户名 + 密码 + 图形验证码 (强制)                            │
│  • 不支持手机号/邮箱自动注册                                    │
│  • 管理员账号必须手动创建                                       │
│                                                                 │
│  安全策略：                                                     │
│  • 强制图形验证码 (每次都显示)                                  │
│  • 失败 3 次锁定 30 分钟                                         │
│  • 仅允许内网访问 (可选配置)                                    │
│  • 操作日志完整记录                                             │
│                                                                 │
│  流程：                                                         │
│  输入用户名 + 密码                                              │
│           ↓                                                     │
│  显示图形验证码                                                 │
│           ↓                                                     │
│  验证验证码                                                     │
│           ↓                                                     │
│  验证用户名 + 密码                                              │
│           ↓                                                     │
│  登录成功？                                                     │
│    ├─ ✅ 是 → 颁发管理员 Token，跳转管理后台                    │
│    └─ ❌ 否 → 记录失败次数                                      │
│              ↓                                                  │
│         失败次数 ≥ 3？                                          │
│           ├─ ✅ 是 → 锁定账号 30 分钟                            │
│           └─ ❌ 否 → 提示"用户名或密码错误"                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 数据库设计 (UUID 版本)

```prisma
// User 表
model User {
  id            String    @id @default(uuid()) @db.VarChar(36)
  phone         String?   @unique
  email         String?   @unique
  username      String?   @unique
  password      String?   // bcrypt 加密，微信登录可为空
  avatar        String?
  nickname      String?
  wechatOpenId  String?   @unique
  status        UserStatus @default(ACTIVE)
  failedAttempts Int      @default(0)
  lockedUntil   DateTime?
  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt
  
  @@index([phone])
  @@index([email])
  @@index([username])
}

enum UserStatus {
  ACTIVE
  DISABLED
  LOCKED
}
```
